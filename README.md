# Corpus_maker
Описание
________

  *  Convert (.doc, .docx, .rtf .odt)  to .txt with Tkinter UI

  *  Конвертирование файлов формата (.doc, .docx, .rtf .odt) в формат .txt с заменой названия файла на аббревиатуру из названий директорий.

  *  Конвертирование .doc возможно в Linux при наличии программы antiword.

  *  Использовался для создания корпуса текстов.
